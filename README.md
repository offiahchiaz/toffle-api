## Description

Toffle API is a Task management service where users can signup and create multiple tasks assigned to a main header.

## Installation

```bash
$ npm install
```

## Pre-requisites

```bash

# Prisma 

# Creates .env file
$ npx prisma init

# migrate your DB
$ npx prisma migrate dev

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```

## Test

```bash

# e2e tests
# Create a .env.test file for your tests
$ npm run test:e2e

```
